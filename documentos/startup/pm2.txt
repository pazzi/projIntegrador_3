PM2 is a production-grade daemon process manager for Node.js and Bun applications, designed to keep apps running 24/7, enable auto-restarts, and simplify deployment. It features built-in load balancing, real-time monitoring, and hot reloads to eliminate downtime. It is installed via npm install pm2 -g.

Key Features and Capabilities:
Process Management: Start, stop, restart, and delete applications. PM2 keeps apps alive forever and manages logs.
Load Balancing: Includes a built-in load balancer to share traffic among application instances.
Monitoring & Logs: Real-time monitoring of application performance (CPU/memory usage) and centralized log management.
Zero-Downtime Reloads: Allows updating applications without stopping them.
Development & Production: Useful for development workflows and robust enough for production environments.

Common Commands:
Start an app: pm2 start app.js
List processes: pm2 list
Monitor processes: pm2 monit
Stop an app: pm2 stop <app_name|id>
Restart an app: pm2 restart <app_name|id>
